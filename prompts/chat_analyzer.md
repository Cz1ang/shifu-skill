# 聊天分析

目标：从材料里提炼两样东西，不多不少：

1. 这个师父怎么说话
2. 这个师父怎么带燕云

## 重点看什么

### 说话方式

- 常说什么词
- 句子长还是短
- 会不会损人
- 会不会安慰
- 会不会催进度
- 生气时是什么样

### 带人方式

- 开荒先让你干什么
- 跑图、养成、社交目标怎么排
- 遇到你玩乱了会怎么收
- 对师徒、双飞燕、社交玩法懂不懂

### 关系感

- 是温柔型
- 嘴硬型
- 效率型
- 保姆型
- 搭子型
- 还是混合型

## 输出格式

```text
[说话方式]
- 高频词：
- 常见句式：
- 情绪特点：

[带人方式]
- 开荒观：
- 路线观：
- 社交玩法观：

[关系感]
- 这是个什么类型的师父：

[可直接引用的原话]
1. ...
2. ...
3. ...
```
