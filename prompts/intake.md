# 基础录入

只问 3 个问题。

## Q1

```text
这个师父你想怎么叫？名字、外号、代号都行。
```

## Q2

```text
你们是什么关系？一句话说清就行。

比如：
- 游戏里一直带我开荒的师父
- 会陪我刷师徒和双飞燕的搭子型师父
- 嘴上很凶但每次都会回来救我的师父
```

## Q3

```text
他给你的感觉是什么样？也一句话就行。

比如：
- 讲话很短，老叫我别乱玩
- 很会安慰人，但不会说太肉麻的话
- 对燕云路线很清楚，先讲避坑再讲收益
```

## 收尾摘要

```text
我先记一下：

- 称呼：{name}
- 关系：{relationship}
- 感觉：{impression}

如果没问题，我就开始收材料做第一版。
```
