# 原材料导入规则

这个 skill 处理的是“蒸馏”，不是“破解”。

## 支持的输入

### 直接可用

- 微信聊天导出的 txt / md / json
- QQ 聊天导出的 txt / html / json
- 手动复制的聊天文本
- 聊天截图
- 群公告、备忘录、语音转写
- 用户手写的人物画像

### 可当补充材料

- 游戏内私聊截图
- 群聊节选
- 师徒任务安排清单
- 他说过的“今日路线”
- 他写过的配装 / 日常建议

## 优先保留的内容

- 指路
- 教学
- 安慰
- 催日常
- 纠错
- 带队决策
- 关于双飞燕 / 师徒 / 社交值的说明
- 对“别碰什么”的强烈判断

## 低价值内容

以下内容除非样本很少，否则权重低：

- 单纯打招呼
- 没信息量的表情包刷屏
- 无上下文的哈哈哈
- 与燕云和关系感都无关的水聊

## 群聊处理

如果来源是群聊：

- 优先提取目标人物自己的发言
- 其他人的话只保留上下文必要部分
- 不要把群里其他人的语气混进目标人格

## 样本不足时的做法

如果没有足够聊天记录：

- 用用户主观描述先做一版
- 明确告诉用户这是“低样本版”
- 后续鼓励追加材料迭代
